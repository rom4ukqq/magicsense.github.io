# magicsense.github.io
Privacy Policy for MagicSense

Last updated: March 2026

MagicSense does not collect, store, or transmit any personal data.

The application operates entirely on the user's local device and does not send any information to external servers.

Data Collection

MagicSense does not collect:

personal information

usage analytics

device identifiers

location data

controller data

Third-Party Services

MagicSense does not use third-party analytics or tracking services.

Permissions

The application may access connected input devices (such as game controllers) only for the purpose of providing its functionality. No data from these devices is transmitted outside the user's computer.

Changes

This privacy policy may be updated in the future. Updates will be reflected on this page.
